快捷键：

插件：

- .ignore  忽略文件

- Translation 翻译插件， 需要自己设置翻译SDK的key 
- Atom Material Icons   一个好看的文件夹主题
- GitToolBox    git 相关插件， 显示历史修改人等
- One Dark Theme  黑色主题
- Tabnine AI Code Completion AI开发助手
